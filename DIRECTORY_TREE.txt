How Agents Use This Structure:
===============================

1. Agent starts
2. Reads: agent-config.json
3. Detects model (e.g., "claude-4")
4. Maps to provider (e.g., "anthropic")
5. Loads skills from: ./anthropic/
6. Loads memory from: ./memory/MEMORY.md
7. Ready to assist with context!


Quick Commands:
===============

# Initialize memory system
./setup.sh

# View configuration
