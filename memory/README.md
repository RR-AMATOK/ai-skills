# Skills Memory System

This directory contains memory files that help improve AI skill performance and results over time.

## Structure

- **MEMORY.md** - Main memory file with key patterns, insights, and preferences
- **patterns.md** - Detailed patterns and best practices (create as needed)
- **debugging.md** - Troubleshooting notes and solutions (create as needed)
- **integrations.md** - Integration patterns and examples (create as needed)

## Purpose

The memory system helps:
- Track what works well for different types of tasks
- Document user preferences and workflow patterns
- Store solutions to recurring problems
- Improve response quality and efficiency over time
- Build on previous experiences across sessions

## Usage

The AI will:
- Consult memory files to provide better assistance
- Update memories based on successful patterns
- Remove or correct outdated information
- Create topic-specific files for detailed notes

## Best Practices

- Keep MEMORY.md concise (< 200 lines for optimal context loading)
- Organize by topic, not chronologically
- Verify patterns across multiple uses before documenting
- Update or remove memories that are incorrect
- Create separate files for detailed notes and link from MEMORY.md
